# Cryst — experimental data (accumulates here)

This crystalliser's OWN measured data lives under its folder, with dignity:
- `solubility/`     measured c_sat(T) runs (per campaign / per sample)
- `psd/`            sieve / laser-diffraction size distributions
- `kinetics-fits/`  G, B0 regressions vs the runs (feeds ../crystallisation)
- `provenance.md`   who measured what, when, how

The unit reads `../crystallisation` (its kinetics) and `../thermoPackage`
(its model); this folder is the lab notebook that those are curated FROM.
