# Choupo Licence Map

Choupo uses separate licences for different kinds of material:

- Source code, cases, tests, scripts, GUI code, and machine-readable examples:
  `GPL-3.0-or-later`; see [`../LICENSE`](../LICENSE).
- Guides/manuals under `docs/`: prose, figures, and explanatory text are
  authored and editorially maintained by Vítor Geraldes and Pedro Mendes
  (except the Developer Guide, authored by Vítor Geraldes alone) under
  `CC-BY-SA-4.0`; see <https://creativecommons.org/licenses/by-sa/4.0/>.
- Code excerpts, Choupo case files, and other machine-readable examples inside
  the guides/manuals: `GPL-3.0-or-later`.
- Curated data under `data/`: licensed per component, with provenance and
  licence notes in the relevant data files/folders.
- Choupo name, logos, wordmarks, and brand assets: not licensed by GPL or
  CC BY-SA; see [`../TRADEMARKS.md`](../TRADEMARKS.md).

The authoritative project notice is [`../NOTICE`](../NOTICE).
